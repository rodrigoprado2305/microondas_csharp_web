# microondas_csharp_web

Ele é uma aplicação simples Web local, para simulação de um display de microondas, usando MVC

## backend

.NetCore 10.0

## Rodando no vsCode

```bash
dotnet clean Rodrigo-MicroOndas.csproj
dotnet clean Rodrigo-MicroOndas.csproj
dotnet run --project Rodrigo-MicroOndas.csproj

dotnet clean MicroOndas.sln
dotnet build MicroOndas.sln
dotnet run --project Rodrigo-MicroOndas.csproj
```
